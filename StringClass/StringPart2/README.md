**This project demonstrates advanced String class methods in Java.  It shows how to split strings using delimiters and regex, match patterns, compare substrings, work with Unicode code points, use utility methods, and format output.**

**split(String regex)** – Splits string by delimiter or regex  
**split(String regex, int limit)** – Splits string into limited parts  
**matches(String regex)** – Tests if string matches regex pattern  
**regionMatches(...)** – Compares substrings (optionally case-insensitive)  
**codePointAt(int index)** – Unicode of character at index  
**codePointBefore(int index)** – Unicode of character before index  
**codePointCount(int start, int end)** – Counts code points in range  
**contains(CharSequence s)** – Checks if substring exists  
**String.join(delimiter, elements)** – Joins strings with delimiter  
**copyValueOf(char[])** – Converts char array to string  
**copyValueOf(char[], start, length)** – Converts part of char array to string  
**String.format(...)** – Formats integers and floating-point numbers with padding, alignment, precision
