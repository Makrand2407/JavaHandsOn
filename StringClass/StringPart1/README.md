**This project demonstrates various String class methods in Java.**
**It shows how to create strings from byte arrays, char arrays, and other strings, and how to use common methods like charAt(), getChars(), getBytes(), toCharArray(), equals(), compareTo(), indexOf(), substring(), concat(), trim(), and more.**

**toCharArray() -	Converts to char array**
**getBytes() -	Converts to byte array**
**length() - 	Returns length of string**
**charAt(int index) - 	Returns character at position**
**equals(Object o) -	Compares content (case sensitive)**
**equalsIgnoreCase(String s) - 	Compares content ignoring case**
**compareTo(String s) -	Lexicographic comparison**
**compareToIgnoreCase(String s) -	Lexicographic comparison ignoring case**
**toLowerCase()	- Converts to lowercase**
**toUpperCase()	- Converts to uppercase**
**substring(int start)	- Returns substring from start**
**substring(int start, int end) -	Returns substring (start to end‑1)**
**concat(String s) -	Concatenates strings**
**trim() -	Removes leading/trailing spaces**
**startsWith(String s)	- Checks prefix**
**endsWith(String s)	- Checks suffix**
**indexOf(...)	- First occurrence of char/substring**
**lastIndexOf(...)	- Last occurrence of char/substring**
