**This project demonstrates various StringBuffer class methods in Java.  
It shows how to create and manipulate mutable strings using operations like append, insert, reverse, replace, delete, and more.**

**length()** – Returns number of characters  
**capacity()** – Returns buffer capacity (default: length + 16)  
**charAt(int index)** – Returns character at position  
**setCharAt(int index, char c)** – Modifies character at position  
**setLength(int newLength)** – Changes length (truncate or pad)  
**append(...)** – Appends values to buffer  
**insert(int offset, String s)** – Inserts substring at position  
**reverse()** – Reverses content  
**replace(start, end, String s)** – Replaces substring  
**delete(start, end)** – Deletes substring  
**deleteCharAt(int index)** – Deletes character at position  
**equals(Object o)** – Reference equality (does not compare content)  
**toString().equals(...)** – Compares actual string content
